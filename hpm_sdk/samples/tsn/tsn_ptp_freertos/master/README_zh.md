# tsn_ptp_freertos_master

## 概述

本示例展示了TSN PTP Master的功能

- Master与Slave通过gPTP协议通讯，从而实现Slave端的时钟同步

## 硬件设置

* 使用USB Type-C线缆连接PC USB端口和PWR DEBUG端口
* 使用以太网线缆连接开发板TSN RGMII端口另一个块开发板的TNS RGMII端口

## 工程配置

- 无
