# I2C_COMPONENT_MASTER_POLLING

## 概述

- i2c_components_master_polling示例工程展示了I2C作为主机，通过polling传输数据的功能，访问具体从机设备的寄存器地址的内存数据以及纯数据访问。
- 示例工程中的具体从机设备的寄存器地址的内存访问，I2C master先写入从机设备的寄存器内存数据，之后再读取从机设备的寄存器内存数据，并对比读取和写入的数据是否一致。
- 示例工程中的纯数据访问，I2C master先写入数据，之后再读取数据，并对比读取和写入的数据是否一致。

## 硬件设置

将两个板子的[I2C引脚](lab_board_app_i2c_pin)相连。

- 先运行slave，再运行master
