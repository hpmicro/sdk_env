.. _hpm_log:

HPM Log
======================

概述
------

hpm_log工程展示了hpm_log组件的使用方法，通过串口作为输出引擎说明硬件引擎的适配方式。相较于传统的printf，hpm_log具非阻塞、带有时间戳等特点易于对应用程序进行调试和分析。

硬件设置
------------

无特殊设置

------------
