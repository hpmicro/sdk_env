.. _ewdg:

EWDG
========

概述
------

本例程展示了EWDG IP的一些典型用例

硬件设置
------------

无特殊设置

运行现象
------------

当工程正确运行后，串口终端会输出如下信息：

.. code-block:: console

   ****************************************************************
   *                                                              *
   *  WDG Example Menu                                            *
   *  1. Test EWDG service                                        *
   *  2. Test EWDG reset                                          *
   *  3. Test EWDG Window Mode                                    *
   ****************************************************************


按对应的数字运行指定的示例。

注意
------

示例2和示例3会触发系统复位，建立在`flash_xip`相关的构建中运行。
