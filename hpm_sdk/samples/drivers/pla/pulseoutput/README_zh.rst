.. _pla_pulse_output:

脉冲输出
========

概述
------

本示例演示了如何将PLA配置成7bit脉冲发生器。通过配置PLA的滤波器、AOI逻辑和触发器，实现一个7位计数器功能，可以产生连续的脉冲输出。

硬件设置
------------

- 连接脉冲输出引脚 :ref:`BOARD_PLA_IO_TRG_NUM <board_resource>` 到示波器

功能描述
------------

- 配置PWM产生基准时钟信号
- 配置PLA的多级滤波器、AOI逻辑和D触发器
- 实现7位计数功能，产生连续的脉冲输出
- PWM周期可通过宏 PWM_PERIOD_IN_MS 配置

------------
