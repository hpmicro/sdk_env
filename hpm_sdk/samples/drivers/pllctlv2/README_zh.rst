.. _pllctlv2:

PLLCTLV2
================

概述
------

**pllctl** 锁相环控制器PLLCTL驱动示例工程展示了通过锁相环控制器，在不同模式下，对系统时钟源进行分频系数配置，达到稳定系统主频及分频或倍频的目的
- 示例代码中，通过配置参考时钟分频系数、反馈分频系数、后分频系数等，实现PLL控制器实现逻辑

- 通过配置PLL实现PLL设置

- 设置系统频率从 **400MHz到1GHz** ，展示实际与预期的主频偏差数值

硬件设置
------------

-  无特殊设置
