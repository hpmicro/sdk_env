.. _gwc:

GWC
======

概述
------

示例工程展示了如何使用 `gwc` 的 `api`, 功能上 `gwc` 对不同分辨率的区域进行crc检查计算，并且对 `GWC` 模块所有功能进行测试

硬件设置
------------

无需特殊的设置

------------
