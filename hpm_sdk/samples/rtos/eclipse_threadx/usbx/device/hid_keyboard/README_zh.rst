USBX 键盘设备
==================

概述
------

该工程实现了一个键盘设备。

硬件设置
------------

- 使用USB Type-C线缆连接PC USB端口和PWR DEBUG端口

- 使用USB Type-C线缆连接PC USB端口和开发板USB0端口

运行现象
------------

当工程正确运行后，程序会循环输入26个小写英文字母和数字1。串口终端会输出如下信息：
