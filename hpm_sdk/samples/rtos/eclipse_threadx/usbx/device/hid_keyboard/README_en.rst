USBX device keyboard
========================================

Overview
--------

The sample implements a keyboard device.

Board Setting
-------------

- Connect a USB port on PC to the PWR DEBUG port on the development board with a USB Type-C cable

- Connect a USB port on PC to one of USB port on the development board with a USB Type-C cable

Running the example
-------------------

When the example runs successfully, it will input lowercase letters and number 1 in a loop, following message is displayed in the terminal:
