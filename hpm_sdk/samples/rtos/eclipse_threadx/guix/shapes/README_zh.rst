.. _shapes:

shapes
============

概述
------

图形切换示例展示了通过Guix创建绘制多种图形绘制、填充的画面。

更多详细信息请参考 `官方 Azure-rtos Guix github <https://github.com/azure-rtos/guix/tree/master/samples>`_

硬件设置
------------

将LCD屏连接到评估板

编译类型
------------

- flash_sdram_xip

- flash_sdram_xip_release

- flash_sdram_uf2

- flash_sdram_uf2_release

运行现象
------------

shapes demo将会显示在LCD屏上
