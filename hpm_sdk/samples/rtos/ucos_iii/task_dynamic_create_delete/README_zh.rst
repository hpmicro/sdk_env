uC/OS-III 任务动态创建删除
====================================

概述
------

该工程演示了uC/OS-III的任务动态创建删除功能。

硬件设置
------------

无特殊设置

运行现象
------------

当工程正确运行后，float task会在数字达到0.11时删除led0_task，在数字达到0.30时，动态创建led0_task，并且led0_task打印的信息的数据会重新从1开始计数，串口终端会输出如下信息：
