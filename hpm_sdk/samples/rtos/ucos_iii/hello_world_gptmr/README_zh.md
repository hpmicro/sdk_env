# uC/OS-III Hello World gptmr

## 概述

该工程演示了使用gptmr作为系统调度时钟源，uC/OS-III的任务调度功能。

## 硬件设置

无特殊设置

## 运行现象

当工程正确运行后，LED会开始闪烁，串口终端会输出如下信息：
```console
Toggle the led pin level.
LED is off.
LED is off.
LED is off.
LED is off.
Toggle the led pin level.
LED is on.
LED is on.
LED is on.
Toggle the led pin level.
LED is off.
LED is off.
LED is off.
LED is off.
```
