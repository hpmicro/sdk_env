# i2c_smbus_master

## 概述

i2c_smbus_master示例工程展示了I2C在master模式作为smbus进行接收和发送数据的功能。在这个示例工程中，SMbus master先发送数据，之后再接收数据，并对比发送和接收到的数据。

## 硬件设置

将两个板子的[I2C引脚](lab_board_app_i2c_pin)相连。

## 运行现象

- 先运行slave，再运行master
