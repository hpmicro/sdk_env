# Eeprom emulation 功能测试

## 概述

EEPROM BASE示例对模拟eeprom的write、read、flush接口进行测试，主要包括以下测试内容：

**写入**

- 写入4笔数据
- 查看返回值是否正确

**读取**

- 读出写入的4笔数据
- 查看返回值是否正确
- 与写入数据进行比较

**数据更新**

- 更新前两笔的数据
- 查看更新情况

**数据整理**

- 整理所设置区域的数据，保留最新数据

## 板级设置

- 无需特殊设置

## 说明
- 一次写入的数据不要超过一个erase_size
- 需在user_config.h文件中设定EEPROM_MAX_VAR_CNT限制写入数据笔数的最大数量，默认为100笔
- 相关写入情况，可修改debug等级查看
