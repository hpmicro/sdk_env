DMAV2循环接收uart数据示例
==================================

概述
------

- DMAV2新增了循环传输功能，本示例工程展示了使用循环传输方式接收UART数据。

- 从DMAV2开始，无需再使用2个链式传输来实现DMA循环传输。

硬件设置
------------

- 使用串口调试助手进行发送、接收数据。

- 串口波特率设置为 ``115200bps`` ，``1个停止位`` ，``无奇偶校验位``

------------

- 使用串口调试助手发送数据，开发板会回发收到的数据，示例如下：
