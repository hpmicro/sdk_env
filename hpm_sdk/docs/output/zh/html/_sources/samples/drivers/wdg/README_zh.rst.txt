WDG
======

概览
------

本示例提供如下功能:
- WDG 中断示例

- WDG 超时复位示例

板级设置
------------

无特殊要求

运行示例
------------

当示例成功运行后，会显示如下菜单：


.. code-block:: console

   ****************************************************************
   *                                                              *
   *  WDG Example Menu                                            *
   *  1. Test WDG interrupt                                       *
   *  2. Test WDG reset                                           *
   *  3. Show valid Interrupt values                              *
   *  4. Show valid Reset values                                  *
   *                                                              *
   *  NOTE:                                                       *
   *    WDG total reset inverval =                                *
   *    interrupt interval + reset interval                       *
   ****************************************************************
