RAMSTERSS example
==================================

概述
------

RAMSTERSS 示例工程对外部的存储器进行压力测试. 主要包含如下测试内容:

- algo_fill_selfaddress

- algo_random

- algo_or

- algo_xor

- algo_and

- algo_sub

- algo_mul

- algo_div

- algo_seqinc

- algo_solidbits

- algo_checkerboard

- algo_blockseq

- algo_walkbits0

- algo_walkbits1

- algo_bitspread

- algo_bitflip

- algo_8bits

- algo_16bits

板级设置
------------

无需特殊设置
