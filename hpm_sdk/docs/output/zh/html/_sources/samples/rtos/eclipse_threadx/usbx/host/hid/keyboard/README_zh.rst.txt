USBX 主机例程-键盘
========================

概述
------

该工程实现了一个可以使用键盘的USB主机。

硬件设置
------------

- 使用USB Type-C线缆线连接PC USB端口和PWR DEBUG端口

- 使用USB Type-C转Type-A线缆线连接开发板USB0端口和键盘。
