uC/OS-III Memory Manage
==============================================

概述
------

该工程演示了uC/OS-III的内存池功能。

硬件设置
------------

无特殊设置

运行现象
------------

当工程正确运行后，串口终端会输出如下信息：

.. code-block:: console

   led0 task get mem
   mem  free block 4
   mem  free block 4
   mem  free block 4
   mem  free block 4
   mem  free block 4
   buf:0x1098a20
   mem  free block 5
   mem  free block 5
   mem  free block 5
