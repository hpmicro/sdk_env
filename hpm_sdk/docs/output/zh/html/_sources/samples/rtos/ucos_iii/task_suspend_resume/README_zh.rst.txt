uC/OS-III 任务暂停和恢复
==================================

概述
------

该工程演示了uC/OS-III的任务暂停和恢复功能。

硬件设置
------------

无特殊设置

运行现象
------------

当工程正确运行后，float task会在数字达到0.11时暂停led0_task，在数字达到0.30时，恢复led0_task，串口终端会输出如下信息：
