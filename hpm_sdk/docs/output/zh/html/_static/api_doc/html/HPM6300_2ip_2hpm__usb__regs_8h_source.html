396 /*
397  * RST (RW)
398  *
399  * RST
400  * Controller Reset (RESET) - Read/Write. Software uses this bit to reset the controller.
401  * This bit is set to zero by the Host/Device Controller when the reset process is complete. Software cannot terminate the reset process early by writing a zero to this register.
402  * Host operation mode:
403  * When software writes a one to this bit, the Controller resets its internal pipelines, timers, counters, state machines etc. to their initial value.
404  * Any transaction currently in progress on USB is immediately terminated. A USB reset is not driven on downstream ports.
405  * Software should not set this bit to a one when the HCHalted bit in the USBSTS register is a zero.
406  * Attempting to reset an actively running host controller will result in undefined behavior.
407  * Device operation mode:
408  * When software writes a one to this bit, the Controller resets its internal pipelines, timers, counters, state machines etc. to their initial value.
409  * Writing a one to this bit when the device is in the attached state is not recommended, because the effect on an attached host is undefined.
410  * In order to ensure that the device is not in an attached state before initiating a device controller reset, all primed endpoints should be flushed and the USBCMD Run/Stop bit should be set to 0.
411  */
