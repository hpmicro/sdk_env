5. HPM5300EVK开发板

5.1. 概述

HPM5300EVK提供了一系列HPM5300系列微控制器特色外设的接口，包括一个ADC输入SMA接口，一个先楫标准的电机控制接口，一个正交编码输出接口，一个RS485/422接口。HPM5300EVK同时集成了1个2x20 pin IO扩展接口，连接了HPM5300 MCU的大部分IO，供用户自由评估。HPM5300EVK为MCU外扩了NOR Flash存储，并集成了板载调试器。
