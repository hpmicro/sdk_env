50.2. QEIV2 ABZ Encoder Oneshot Mode

50.2.1. 概述

Qeiv2 ABZ Encoder Oneshot Mode工程主要演示配置ABZ编码器并获取其单次的数据，然后通过串口终端将数据打印出来。

50.2.2. 配置

  • 安装串口终端，查看板子信息并配置串口终端参数

  • 电机使用的是雷赛智能 BLM57050-1000 无刷电机，电机具体参数请参考雷赛智能官网。

  • 点击查看DRV-LV50A-MP1907电机驱动板章节并进行配置

50.2.3. 运行现象

  • 上电后，打开串口终端，打印启动信息。
