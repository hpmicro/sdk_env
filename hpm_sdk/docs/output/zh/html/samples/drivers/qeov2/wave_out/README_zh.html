51.4. QEOV2_WAVE_OUT

51.4.1. 概述

QEO_WAVE_OUT示例工程展示QEO根据得到的位置信息，生成弦波, 可以通过PWM输出占空比可变的PWM。

51.4.2. 注意

QEO支持输出3路弦波信号。

51.4.3. 运行现象
