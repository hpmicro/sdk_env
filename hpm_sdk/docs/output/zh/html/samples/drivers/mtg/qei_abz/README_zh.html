35.2. MTG QEI ABZ 例程

35.2.1. 概述

MTG QEI ABZ 例程演示了使用QEI ABZ类型编码器作为数据输入，经过MTG的滤波器进行处理，以及经过轨迹预测模块对其进行时间补偿的功能。例程输出QEI的原始数据和经过MTG处理后的数据，并输出两者之间的差值。

35.2.2. 硬件设置

无特殊设置
