61. SYNT

61.1. 概述

synt同步定时器驱动示例工程展示了通过设置同步定时器模块，实时获取同步计时器的计数信息

  • 示例代码中，初始化同步定时器，装置计数重载值，循环读取同步计时器中的计数值，循环次数设置为100次

61.2. 硬件设置

  • 无特殊设置

61.3. 运行现象
