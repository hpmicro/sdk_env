37.2. SPI SDCARD FATFS

37.2.1. 概览

本示例使用SPI接口操作sdcard，并且使用fatfs文件系统。测试需要格式化SD卡为FAT32，提供如下功能:

  • 创建hello.txt

  • 从hello.txt读出一行文本

  • 创建并删除文件夹

  • 大文件测试

37.2.2. 板级设置

  • SPI引脚根据板子型号查看具体信息

  • SPI引脚对应好SD卡(模块)引脚
