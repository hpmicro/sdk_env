27.8. 差错检测和修正的算法ECC-DSA

27.8.1. 概述

ecc_dsa_example示例工程展示了差错检测和修正的算法ECC的实现。

  • 此工程的差错检测和修正的算法采用ECC DSA完成模块加解密测试。

27.8.2. 硬件设置

  • 无特殊设置
