40.10. SHA-HMAC算法

40.10.1. 概述

sha256_hmac_example 示例工程展示了SHA算法-HMAC密钥报文完整性验证方法的实现。 - 此工程的算法采用HMAC SHA256法完成模块加解密测试。

40.10.2. 硬件设置

  • 无特殊设置
