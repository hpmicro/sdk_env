21.4. i2c_interrupt_slave

21.4.1. Overview

The i2c_interrupt_slave sample project shows that I2C sends/receives data with interrupt way at slave mode. In this sample, then I2C slave will sent out received data.

21.4.2. Board Setting

connect I2C Pins on the two boards。

21.4.3. Running the example

  • Run slave first, and then run master.

  • When the example runs successfully, the log would be seen on the terminal like:

I2C interrupt slave example, I2C will sent back the received data
