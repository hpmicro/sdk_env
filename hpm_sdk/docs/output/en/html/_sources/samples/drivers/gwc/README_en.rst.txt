.. _gwc:

GWC
======

Overview
--------

The example project demonstrates how to use `gwc`'s `api`. In terms of functionality, `gwc` performs CRC checks on regions of different resolutions and tests all features of the `GWC` module.

Hardware Configuration
----------------------

No special configuration is required.

Running the Program
-------------------

If the example runs successfully, the serial output will be as follows:


.. code-block:: console

   gwc start..
