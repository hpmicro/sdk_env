.. _vsc_demo:

VSC demo
================

Overview
--------

This example provides a way for software to inject numerical values to determine the correctness of VSC calculation results.
Non software injection method, please reference to the sample: samples/motor_ctrl/bldc_foc_hw

Board Setting
-------------

No special settings are required

Running the example
-------------------

When the example runs successfully, the board shows the following menu:


.. code-block:: console

   VSC software inject sample
   VSC software inject calculation finish! Convert Count: 1080
   VSC software inject calculation all matched, success!
