DMAV2 Circularly Receives Uart Data
======================================================================

Overview
--------

- DMAV2 has added circularly transfer function, This example project shows the use of circularly transfer to receive UART DATA.

- Starting from DMAV2, there is no need to use two chain transfers to achieve DMA circularly transfer.

Board Setting
-------------

- Use the serial port debugging assistant to send and receive data

- The serial port is set to ``115200bps`` , ``1 stop bit`` , ``no parity bit``

Running the example
-------------------

- Use the serial port debugging assistant to send data, and the development board will send back the received data, as shown in the following example
