Device Dual Port
================================

Overview
--------

This example project shows two USB ports as device. USB0 is defined CDC_ACM class, USB1 is defined MSC class.

- PC sees a serial port and a usb disk via Device Manager

Board Setting
-------------

- Connect a USB port on PC to the PWR DEBUG port on the development board with a USB Type-C cable

- Connect a USB port on PC to USB0 port on the development board with a USB Type-C cable

- Connect a USB port on PC to USB1 port on the development board with a USB Type-C cable

Running the example
-------------------

- Download the program and run.

- When connect a USB port on PC to USB0 port on the development board, PC can automatically recognize and install the USB to serial port driver and enumerate a device with a com port.

    - Open the "Serial Port Debugging Assistant" software, check DTR, select the enumerated com port, and click to open the serial port

    - After opening the serial port, the "Serial Port Debugging Assistant" will receive the following characters:

- When connect a USB port on PC to USB1 port on the development board, PC can automatically recognize and enumerate a USB disk device.

    - You can copy a file to the USB disk, and then copy it from the USB disk.
